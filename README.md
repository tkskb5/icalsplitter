# iCAL Splitter

大容量iCALファイルをGoogleカレンダーにインポート可能なサイズに分割するWebツール。

## 特徴

- 🔒 **プライバシー優先**: ファイルはブラウザ内で処理。サーバーには送信されません
- ✂️ **3つの分割方法**: サイズ / 期間 / 年別
- 🧹 **Googleカレンダー互換モード**: Apple固有プロパティを除去
- 📦 **ZIPダウンロード**: 分割ファイルを一括ダウンロード

## 使い方

1. iCAL(.ics)ファイルをアップロード
2. 分割方法を選択
3. ダウンロード
4. Googleカレンダーにインポート

## 技術スタック

- HTML / CSS / JavaScript (バニラ)
- JSZip (ZIPダウンロード用)
- GitHub Pages でホスティング
